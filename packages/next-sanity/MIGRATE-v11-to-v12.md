## Migrate

### Minimum required `sanity` is now `5.0.0`

Upgrade to the latest v5 stable using the following command:

```bash
npm install sanity@latest --save-exact
```

[Read the changelog for v5](https://www.sanity.io/docs/changelog/fd3ab62e-9264-4e7b-825a-fd4f99abd481)

### Minimum required `react` and `react-dom` is now `19.2.3`

[If you're still on React 18, you need to upgrade to React 19.2.3 or later.](https://react.dev/blog/2024/04/25/react-19-upgrade-guide)
